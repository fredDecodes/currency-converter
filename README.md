# Currency Converter Web App

This project is a functional currency converter web application designed to convert values between different currencies in real time. The application demonstrates practical front-end development concepts by integrating external API data with dynamic user interactions. Core concepts implemented in the project include event handling, DOM manipulation, and asynchronous JavaScript functionality using `fetch` for API communication.

# Features

## 1. Currency Conversion Computation
  Performs real-time conversion between selected currencies based on exchange rate data.

## 2. Dynamic Display Updates
  Updates conversion results instantly based on user input and selected currencies.

## 3. Clear Functionality
  Allows users to reset input fields and conversion results.

## 4. Delete Capability
  Enables users to remove entered values for correction and recalculation.

## 5. API Integration
  Fetches live or external exchange rate data using JavaScript’s `fetch` functionality.

# Tech Stack

1. HTML – Defines the structure of the web application interface
2. CSS – Handles styling, layout, and responsive design
3. JavaScript – Implements application functionality
  - Event handling for user interactions
  - DOM manipulation for dynamic UI updates
  - Asynchronous API calls using `fetch`
  - Currency conversion logic and calculations

# Installation & Setup

- Clone or download the project repository to your local machine.
- Open the project directory.
- Ensure internet access is available for API requests.
- Open the main HTML file in a modern web browser.

No additional frameworks or external dependencies are required beyond the exchange rate API service used by the application.

# Project Structure

- index.html
  - Contains the application layout, input fields, currency selectors, and result display sections.

- styles.css
  - Defines the visual styling, responsiveness, and user interface presentation.

- script.js
  - Implements the main application logic, including:

    - Event listeners for input and button interactions
    - Currency conversion calculations
    - API requests using `fetch`
    - DOM updates for displaying conversion results and feedback

The project structure follows a modular and maintainable approach, separating interface design, styling, and application logic into distinct files.
